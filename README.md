# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/forgotTheAwait/Landing-page.git
```

Move into the project

```bash
cd Landing-page
```

Install dependencies

```bash
npm install
```

Start the development server

```bash
npm run dev
```

Open

```
http://localhost:3000
```
